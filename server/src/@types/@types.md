# @types

## English

This folder contains custom TypeScript type declarations for the project. It is used for global types, module augmentation, or shared declaration files.

## Tiếng Việt

Thư mục này chứa các khai báo type TypeScript tùy chỉnh của project. Nó được dùng cho global type, mở rộng module hoặc các file khai báo dùng chung.
