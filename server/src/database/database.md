# database

## English

This folder contains database setup files. It is responsible for database module registration, providers, connection configuration, and shared database utilities.

## Tiếng Việt

Thư mục này chứa các file thiết lập database. Nó chịu trách nhiệm đăng ký database module, provider, cấu hình kết nối và tiện ích database dùng chung.
