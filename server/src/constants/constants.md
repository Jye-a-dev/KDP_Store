# constants

## English

This folder contains shared constant values. Constants keep repeated static values in one place so they can be reused safely across the project.

## Tiếng Việt

Thư mục này chứa các hằng số dùng chung. Constant giúp gom các giá trị tĩnh lặp lại vào một nơi để tái sử dụng an toàn trong toàn project.
