# config

## English

This folder contains application configuration files. It is used for environment configuration, app settings, and centralized configuration values.

## Tiếng Việt

Thư mục này chứa các file cấu hình của ứng dụng. Nó được dùng cho cấu hình môi trường, thiết lập app và các giá trị cấu hình tập trung.
