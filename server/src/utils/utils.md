# utils

## English

This folder contains shared utility functions. Utilities should hold small reusable helpers that do not belong to a specific feature module.

## Tiếng Việt

Thư mục này chứa các hàm tiện ích dùng chung. Utility nên chứa những helper nhỏ có thể tái sử dụng và không thuộc riêng một module tính năng cụ thể.
