# swagger

## English

This folder contains Swagger/OpenAPI configuration. It centralizes API documentation setup and shared documentation options for the server.

## Tiếng Việt

Thư mục này chứa cấu hình Swagger/OpenAPI. Nó tập trung phần thiết lập tài liệu API và các tùy chọn tài liệu dùng chung cho server.
