# validators

## English

This folder contains custom validation helpers. Validators are used to check input values, request parameters, or domain-specific constraints.

## Tiếng Việt

Thư mục này chứa các helper validate tùy chỉnh. Validator được dùng để kiểm tra dữ liệu đầu vào, tham số request hoặc các ràng buộc nghiệp vụ cụ thể.
