# src

## English

This folder contains the main application source code. It includes app bootstrap files, feature modules, shared utilities, configuration, constants, validators, database setup, and Swagger setup.

## Tiếng Việt

Thư mục này chứa mã nguồn chính của ứng dụng. Nó bao gồm các file khởi tạo app, module tính năng, tiện ích dùng chung, cấu hình, hằng số, validator, thiết lập database và thiết lập Swagger.
