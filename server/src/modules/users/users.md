# users

## English

This folder is reserved for the users feature module. It should contain user-related controllers, services, DTOs, entities, interfaces, repositories, and Swagger documentation.

Thư mục này dành cho module tính năng users. Nó nên chứa controller, service, DTO, entity, interface, repository và tài liệu Swagger liên quan đến người dùng.
