# modules

## English

This folder contains feature modules. Each module should group its own controller, service, DTOs, entities, interfaces, repositories, and documentation files.

## Tiếng Việt

Thư mục này chứa các module tính năng. Mỗi module nên gom controller, service, DTO, entity, interface, repository và file tài liệu riêng của module đó.
