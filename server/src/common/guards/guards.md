# guards

## English

This folder contains guards. Guards decide whether a request is allowed to continue based on authentication, authorization, roles, or other access rules.

## Tiếng Việt

Thư mục này chứa các guard. Guard quyết định request có được tiếp tục hay không dựa trên xác thực, phân quyền, vai trò hoặc các quy tắc truy cập khác.
