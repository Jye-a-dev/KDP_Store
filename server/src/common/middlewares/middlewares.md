# middlewares

## English

This folder contains middlewares. Middlewares run before route handlers and are commonly used for request preprocessing, logging, context setup, or request enrichment.

## Tiếng Việt

Thư mục này chứa các middleware. Middleware chạy trước route handler và thường dùng để tiền xử lý request, logging, thiết lập context hoặc bổ sung dữ liệu cho request.
