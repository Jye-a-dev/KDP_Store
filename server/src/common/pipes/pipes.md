# pipes

## English

This folder contains pipes. Pipes are used to transform input data and validate request payloads before they reach controllers or services.

## Tiếng Việt

Thư mục này chứa các pipe. Pipe được dùng để biến đổi dữ liệu đầu vào và validate payload trước khi dữ liệu đi vào controller hoặc service.
