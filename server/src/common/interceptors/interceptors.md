# interceptors

## English

This folder contains interceptors. Interceptors can transform responses, wrap returned data, add logging, measure execution time, or apply cross-cutting behavior.

## Tiếng Việt

Thư mục này chứa các interceptor. Interceptor có thể biến đổi response, bọc dữ liệu trả về, thêm logging, đo thời gian xử lý hoặc áp dụng logic dùng chung.
