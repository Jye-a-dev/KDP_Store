# server

## English

This folder is the root of the backend server project. It contains the NestJS application source code, tests, configuration files, package metadata, and dependency lock files.

## Tiếng Việt

Thư mục này là thư mục gốc của project backend server. Nó chứa mã nguồn ứng dụng NestJS, test, file cấu hình, thông tin package và file khóa dependency.
