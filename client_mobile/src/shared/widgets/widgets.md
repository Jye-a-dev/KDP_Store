# widgets

## English
This folder contains generic reusable widgets and components utilized across the entire project (e.g. custom buttons, loading spinners, input fields).

## Tiếng Việt
Thư mục này chứa các widget và component dùng chung cho toàn bộ dự án (như nút nhấn tùy chỉnh, vòng xoay tải dữ liệu, ô nhập liệu thông dụng).
