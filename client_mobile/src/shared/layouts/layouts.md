# layouts

## English
This folder contains page layout wrapper templates (e.g. public layouts with headers/footers, dashboard layouts with sidebars/headers) that ensure consistent page formatting.

## Tiếng Việt
Thư mục này chứa các mẫu layout bọc ngoài trang (như public layout kèm header/footer, dashboard layout kèm sidebar/header) giúp thống nhất bố cục giao diện.
