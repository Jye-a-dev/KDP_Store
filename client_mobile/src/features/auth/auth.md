# auth

## English
This folder contains the authentication feature. It should hold login, register, session, and account access UI or logic related to identifying and authorizing users.

## Tiếng Việt
Thư mục này chứa feature xác thực. Nó nên bao gồm giao diện hoặc logic cho đăng nhập, đăng ký, phiên đăng nhập và quyền truy cập tài khoản của người dùng.
