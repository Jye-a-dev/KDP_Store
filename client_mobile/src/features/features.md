# features

## English
This folder groups the app by business features. Each feature owns its pages, widgets, controllers, and related UI logic so the project stays modular and easy to expand.

## Tiếng Việt
Thư mục này chia ứng dụng theo từng feature nghiệp vụ. Mỗi feature tự quản lý pages, widgets, controllers và logic giao diện liên quan để project rõ ràng, dễ mở rộng.
