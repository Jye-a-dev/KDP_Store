# public

## English
This folder contains the public facing feature area, containing landing pages, promotional info, and information visible without authenticating.

## Tiếng Việt
Thư mục này chứa khu vực chức năng công khai (public), gồm các trang giới thiệu, thông tin quảng bá và nội dung có thể xem được mà không cần đăng nhập.
