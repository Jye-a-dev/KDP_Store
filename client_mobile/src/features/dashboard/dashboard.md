# dashboard

## English
This folder contains the dashboard feature. It should hold analytics, admin screens, user profiles, and post-login UI and business logic.

## Tiếng Việt
Thư mục này chứa feature dashboard. Nó nên bao gồm giao diện phân tích, màn hình quản lý, hồ sơ người dùng và các giao diện hoặc logic sau khi đăng nhập.
