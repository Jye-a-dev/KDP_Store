# theme

## English
This folder contains global visual styling for the Expo React Native app, including colors, typography, background styling, component defaults, and other shared theme configuration.

## Tiếng Việt
Thư mục này chứa cấu hình giao diện toàn cục của ứng dụng Expo React Native, gồm màu sắc, kiểu chữ, nền màn hình, style mặc định của component và các thiết lập theme dùng chung.
