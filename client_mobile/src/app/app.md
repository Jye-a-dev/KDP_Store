# app

## English
This folder contains the application shell for the Expo React Native project. It wires together global app configuration such as routing, theme setup, and the root navigator used by the UI.

## Tiếng Việt
Thư mục này chứa phần khung chính của ứng dụng Expo React Native. Nó kết nối các cấu hình toàn cục như định tuyến, theme và navigator gốc dùng để chạy giao diện.
