# core

## English
This folder contains the core logic of the application, including constants, API services, type definitions, and helper utilities shared across features.

## Tiếng Việt
Thư mục này chứa logic cốt lõi của ứng dụng, gồm hằng số, api service, định nghĩa kiểu dữ liệu (types) và các hàm tiện ích (utils) dùng chung.
