# services

## English
This folder contains external API service clients, local storage services, database adapters, and other backend connectivity utilities.

## Tiếng Việt
Thư mục này chứa các client gọi API bên ngoài, dịch vụ lưu trữ cục bộ, adapter cơ sở dữ liệu và các tiện ích kết nối backend khác.
