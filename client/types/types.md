# `types`

## Purpose / Mục đích
- English: This folder defines shared TypeScript types, interfaces, and contracts used across the project.
- Tiếng Việt: Thư mục này định nghĩa các kiểu TypeScript, interface và contract dùng chung trong toàn dự án.

## What should live here / Nên đặt gì ở đây
- English: DTO types, API response types, model interfaces, form payload types, and shared prop contracts.
- Tiếng Việt: Kiểu DTO, kiểu response API, interface model, kiểu payload cho form và contract props dùng chung.
- English: Types reused by multiple folders to keep the project consistent.
- Tiếng Việt: Các kiểu được nhiều folder dùng lại để giữ tính nhất quán trong dự án.

## What should not live here / Không nên đặt ở đây
- English: Runtime validation code, service implementations, or component rendering logic.
- Tiếng Việt: Mã validation lúc chạy, phần cài đặt service hoặc logic render component.

## Example / Ví dụ
- English: `auth.type.ts`, `user.type.ts`, `common.type.ts`
- Tiếng Việt: `auth.type.ts`, `user.type.ts`, `common.type.ts`
